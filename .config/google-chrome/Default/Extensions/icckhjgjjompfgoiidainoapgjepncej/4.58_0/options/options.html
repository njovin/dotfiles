Problems?
If you see the song information in the extensions popup window, but the media keys aren't working, go to chrome://extensions and click the Keyboard Shortcuts button at the bottom to make sure they are set.

Make sure you don't have any competing extensions stealing the keys, like Google Play Music.

If you're on Windows, Microsoft Keyboard and Mouse center or Logitech SetPoint can interfere with media keys. Uninstall them if possible.

If it still doesn't work, or the site you use isn't supported or is broken, email us at hello@sway.fm for help.
