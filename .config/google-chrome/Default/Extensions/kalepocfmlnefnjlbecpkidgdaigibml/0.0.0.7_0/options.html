Configure your Twitter™ accounts here.

Name : This will show up for selecting the Twitter™ account when QuickTweet extension is clicked. Your Twitter™ id will show up, if the name is not provided.
Email : Your Twitter™ account username or email.
Password : Your Twitter™ account password (Optional). if password is not provided here, extension will stop the auto login process for you to enter the password.
Use Current tab : When enabled, the extension will load pages in the current active tab.
Click "Save" to apply the changes made. More information is available here.
All the information provided here will be stored in your local computer. The passwords are stored encrypted using advanced encryption methods, ideally accessible only by the extension.
